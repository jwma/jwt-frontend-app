# Vue-JWT

> Vue + VueRouter + Axios实现一套 JWT 用户身份验证的前端应用

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
